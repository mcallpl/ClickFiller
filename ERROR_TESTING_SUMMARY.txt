THE BLOCKER
══════════════════════════════════════════════════════════════════════════

Error:    "process is not defined"
Location: src/app-init.js (lines 16, 23)
Cause:    Using process.env.DEBUG without Vite configuration
Impact:   Form inputs not accessible, validation untestable
Frequency: 100% of page loads
Severity: 🔴 CRITICAL

ROOT CAUSE CODE:
────────────────
// src/app-init.js
if (process.env.DEBUG) {
  // Debugging setup
}

PROBLEM: 'process' is a Node.js global, not available in browsers

SOLUTION (Pick One):
──────────────────

Option 1 - Update vite.config.js (RECOMMENDED):
  define: {
    'process.env': JSON.stringify(process.env)
  }

Option 2 - Use Vite-compatible alternative in app-init.js:
  if (import.meta.env.DEBUG) {
    // Same debugging setup
  }

FIX EFFORT: ~5 minutes
